Brand Guide

v1.0

Logo

Primary logotype and shield mark. Use SVG where possible. Maintain clear space equal to the height of the “L”.

Do: keep adequate contrast, leave clear space. Don’t: stretch, add effects, or recolor outside the palette.

Colors

Primary: #172738

Shades: 700 #101b27 · 600 #13212f · 400 #394755 · 200 #68727d

Status: Success #1F7A5D · Warning #B88700 · Error #B00020

Typography

Use the following stack:

Poppins, Inter, Segoe UI, system-ui, -apple-system, Arial, sans-serif

Weights: 400/500/600/700. Base size: 16px, line-height 1.5.

Icons & PWA

Favicons and app icons are included. For Safari pinned tabs, the SVG must be monochrome; color is set in HTML via color="#111111".

<link rel="mask-icon" href="https://assets.lucidseal.org/brand/assets/app/safari-pinned-tab.svg" color="#111111">

Badge Usage

Sizes: Inline 14px (for text), UI 24–32px (buttons, headers), Marketing 128–256px. State colors must pass AA at small sizes.

For color variants, use status tokens from brand/tokens.json.
